# Définition des Endpoints:<br>
on a décidé définir 6 Endpoints prioritaires

## getAllCommandesNonLivres: <br>
    Récupérer toutes les commandes avec le status NonLivrés ou annulé, 
    l'algorithme de création des tournées prendra ces commandes pour ensuite 
    créer des tournées 
## getNombreDeCommandesNonLivrés
    Récupérer le nombre de commandes avec le status non livrés 

## getNombreEquipes
    Récupérer le nombre des équipes disponibles 

## getNombreHeuresMax
    Récupérer le nombre d'heures max définie 

## createTournee
    suite au calcule des tournées via les algorithme et validation du planificateur , 
    les tournées validés vont être stockés dans la base de données 

## getAllTournéesByDate
    Récupérer toutes les tournées stockés dans la base de données d'une date spécifique
